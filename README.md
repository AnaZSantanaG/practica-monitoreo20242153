# Práctica: Monitoreo con Prometheus y Grafana
Este proyecto configura un entorno de monitoreo básico usando Docker.

## Instrucciones para ejecutar:
1. Asegúrate de tener Docker instalado.
2. Ejecuta: `docker-compose up -d`
3. Accede a **Prometheus**: http://localhost:9090
4. Accede a **Grafana**: http://localhost:3000 (Usuario: `admin`, Clave: `admin`)
